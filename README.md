# CICD-Kubernetes-Project
A production-ready CI/CD platform showcasing automated build, test, Dockerization, and Kubernetes deployment using GitHub Actions, Docker, and AWS ECR.

# DevOps CI/CD Kubernetes Project

This project demonstrates a full CI/CD pipeline using:

- GitHub Actions
- Docker
- AWS ECR
- Kubernetes (Minikube)

Features:
- Automated build
- Docker image creation
- Container registry push
- Kubernetes deployment
